# Car-Rental

Aplikacja wykonywana w ramach pracy inżynierskiej.  
Temat: Aplikacja wspierająca pracę wypożyczalni samochodów.


#### Wykorzystane technologie:

- Java
- Tomcat
- Project Lombok
- Maven
- Spring Framework
- Spring Boot
- Spring Security
- MySQL
- JPA - Hibernate
- JSP, JSTL
- Frontend
  - Bootstrap 4
  - jQuery
  - Html
  - SCSS, CSS
  - bcSwipe.js
- Środowisko programistyczne: IntelliJ IDEA
